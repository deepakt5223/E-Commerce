# 🛒 E-Commerce Data Analysis Project

## 📌 Overview

This project focuses on analyzing an e-commerce dataset using **SQL and Python**. The goal is to extract meaningful insights related to customers, orders, revenue, product_ratings and product performance.

---

## 🎯 Objectives

- Analyze customer purchasing behavior
- Identify top-performing products, category and brand
- Track revenue trends
- Detect good vs bad customers
- Perform data cleaning and preprocessing
- City-wise revenue.

---

## 🧰 Tools & Technologies Used

- **Python** (Pandas, NumPy)
- **SQL** (MySQL)
- **matplot.pyplot and seaborn (for visualization)**
- **Jupyter Notebook**

---

## 📂 Dataset

- Source: Kaggle(e-commerce dataset)
- Contains information about:
  - Customers
  - Orders
  - Products
  - Ratings
  - Cities
  - Revenue

---

## 🔍 Key Analysis Performed

### 1. Data Cleaning

- Checked null values
- checked duplicates
- Checked data consistency

### 2. Customer Analysis

- Identified good and bad customers
- Purchase frequency analysis

### 3. Product Analysis

- Top-selling products
- Low-performing products

### 4. Revenue Analysis

- Total revenue calculation
- Revenue trends over time

### 5. City-wise Analysis

- Orders distribution by city
- Revenue contribution by location
